# Library Management System

Um sistema simples de gerenciamento de livros desenvolvido em Python durante meus estudos.

## Funcionalidades

- Cadastrar livros
- Listar livros em ordem alfabética
- Buscar livros
- Editar livros
- Remover livros
- Impedir livros duplicados
- Contar a quantidade de livros cadastrados

## Tecnologias

- Python 3

## Como executar

1. Clone o repositório.
2. Execute o arquivo `main.py`.

```bash
python main.py
```

## Objetivo

Este projeto foi desenvolvido para praticar:

- Funções
- Modularização
- Manipulação de arquivos
- Tratamento de exceções
- Organização de código
